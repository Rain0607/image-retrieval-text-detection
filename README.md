# 校园图像检索与文字检测项目
## 项目简介
本项目包含两项自监督深度学习任务，全部基于校园实景图片完成：
1. **图像检索任务**：采用SimCLR对比学习，在无标签数据集上训练ResNet50主干网络，实现以图搜图，使用P@K指标完成检索精度定量评估。
2. **场景文字检测任务**：不依赖人工标注，利用图片亮度阈值自动生成伪标签，训练简易分割网络预测文字区域；生成检测效果图，人工分析漏检与误检问题。

## 任务一：无监督图像检索（SimCLR对比学习）
### 实验目标
1. 在无标签图库上，使用NT-Xent对比损失训练ResNet50网络，学习图片特征表达。
2. 提取归一化特征，计算余弦相似度，完成相似图片检索。
3. 计算Precision@20 / Precision@40 / Precision@60，绘制12个地标的P@K精度曲线。

### 运行命令
```bash
python train_retrieval.py
```

### 输出内容
1. 模型权重文件：`self_trained_resnet.pth`
2. 每一类别的P@K折线图，自动保存至`result_fig`文件夹
3. 控制台打印各类别精确率数值

## 任务二：自监督场景文字检测
### 实验目标
1. 无任何JSON文字标注，依靠图像灰度阈值自动生成文字二值伪标签。
2. 搭建轻量化卷积分割网络，训练模型预测文字区域并绘制矩形框。
3. 每个类别选取2张测试图片，共生成24组原图+检测效果图，人工分析误检、漏检情况。

### 第一步：训练分割网络
```bash
python train_detect.py
```
- 程序自动校验图片文件，跳过破损图片，防止程序崩溃。
- 自动提取高亮区域作为文字伪标签，构建分割数据集。
- 完成训练，保存权重`text_self_train.pth`，并为训练集图片生成带框效果图。

### 第二步：在query测试集上推理测试
```bash
python test_detect.py
```
- 严格保持网络结构与训练代码一致，加载权重完成预测。
- 文字检测效果图自动输出至`query_test_output`文件夹。

### 第三步：拼接成对对比图
```bash
python merge_pic.py
```
将原图与文字检测效果图左右拼接，用于后续人工主观评测。

## 实验结果分析
### 图像检索任务
模型通过对比学习学习图片视觉特征，P@K精度可以直观衡量校园风景图片的检索排序准确度。

### 文字检测任务
模型能够大致识别石碑上的文字区域。但由于仅依靠亮度生成伪标签，大量高对比度背景（树缝、建筑窗户、阴影边缘、路面纹理）会被误识别为文字，造成检测框过多。可以通过调高预测阈值、过滤小面积候选框来减少误检。

## 注意事项
1. 代码内置文件校验逻辑，会自动跳过损坏图片，避免OpenCV读取报错。
2. Windows系统下若大量图片读取失败，可改用PIL库加载图片。
3. 两个模型相互独立：ResNet50用于图像检索，轻量CNN用于文字分割检测。
4. 全程均为自监督训练，全程不需要人工标注数据。
